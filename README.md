# Laravel Repository CRUD

## Follow step by step:


### copy .env.example file to .env

```
cp .env.example .env
```

### install vendor files

```
composer install
```

### install nodemodules files

```
npm install
```

### minify css or js

```
npm run dev
```

### genrate app key

```
php artisan key:genrate
```
### link storage folder to public folder

```
php artisan storage:link
```
